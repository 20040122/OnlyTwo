- 注册：
flowchart LR
      A[用户输入邮箱和密码] --> B[数据库校验]
      B --> C{是否有效}
      C -- 否 --> D[返回错误提示]
      C -- 是 --> E[检查邮箱是否已注册]
      E --> F{是否已存在}
      F -- 是 --> G[提示邮箱已注册]
      F -- 否 --> H[创建账号]
      H --> I[发送验证邮件]
      I --> J[用户完成邮箱验证]
      J --> K[注册成功]
      K --> L[返回登录界面]

- 登录：
flowchart LR
      A[用户输入邮箱和密码] --> B[系统校验输入]
      B --> C{是否有效}
      C -- 否 --> D[返回错误提示]
      C -- 是 --> E[提交登录请求]
      E --> F[Supabase Auth 校验账号密码]
      F --> G{是否通过}
      G -- 否 --> H[提示邮箱或密码错误]
      G -- 是 --> I{邮箱是否已验证}
      I -- 否 --> J[提示先验证邮箱]
      I -- 是 --> K[登录成功]
